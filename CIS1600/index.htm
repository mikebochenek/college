270-1600 Foundations of Programming

All the assignments are in JAVA and compile using Sun's JDK 1.1.6 or later.

Assignment #1: AssignmentOne.java - My very first programming assignment at University.  It's embarrasingly simple, it simply asks the user to enter a number in pounds and converts it to kilograms.

Assignment #2: Assignment2.java - This program will ask the user to enter time and distance run, and will output various statistics, such as average speed, time required for each km/mile run, etc.

Assignment #3: Personnel.java - This program reads information about employees and stores it in an array of structures and displays a menu.  Using the menu, the user can print hightest/lowest paid employees, print a list of all employees, etc.

Assignment #4: Draw.java - This one is a little bit more fun.  It is a Java Applet that allows you to draw simple objects using the mouse.  You can draw rectangles, circles, lines, and points in black.  Click here if you want to run it.
