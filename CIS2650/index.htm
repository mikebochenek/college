270-1700 Introduction to Computing & Information Science

All the assignments are in C and compile using GCC.

Assignment #1: a1.c - This program reads information on students from a data file, and displays a menu.  It allows users to inquire about various statistics and list all the students.

Assignment #2: a2.c - This program reads a finite state automaton definition from a file, and then checks if various strings pass or fail.
